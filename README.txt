Чат-приложение на C++
Участники команды
Участник 1: [Иван]

Идея решения — создать простое консольное чат-приложение на C++, которое позволяет пользователям регистрироваться, 
авторизоваться и обмениваться сообщениями. Приложение реализует основные функции чата, включая отправку сообщений как конкретному пользователю, так и всем пользователям одновременно.

Решение было разделено на несколько классов:

User — класс для представления пользователя, содержащий логин, пароль и имя.
Message — класс для представления сообщения, которое отправляется от одного пользователя к другому.
Chat — главный класс, который управляет пользователями, регистрацией, авторизацией и хранением сообщений.
main — главный файл программы, который управляет взаимодействием с пользователем через консоль, а также выполняет все необходимые операции.
Приложение использует объектно-ориентированный подход, где каждый класс отвечает за свою часть функционала. Пользователи могут отправлять сообщения конкретным получателям или всем одновременно.

Класс User

Поля:
std::string username — логин пользователя.
std::string password — пароль пользователя.
std::string name — полное имя пользователя.

Методы:
User(std::string uname, std::string pass, std::string fullname) — конструктор, создающий объект пользователя.
void display() const — метод для вывода информации о пользователе.


Класс Message

Поля:
User* sender — отправитель сообщения.
User* recipient — получатель сообщения.
std::string content — текст сообщения.

Методы:
Message(User* sender, User* recipient, const std::string& msg) — конструктор для создания сообщения.
void display() const — метод для вывода сообщения.
Класс Chat

Поля:
std::unordered_map<std::string, User*> users — контейнер для хранения пользователей по их логину.
std::vector<Message> messages — контейнер для хранения всех сообщений.

Методы:
void registerUser(const std::string& username, const std::string& password, const std::string& name) — метод для регистрации пользователя.
User* authenticateUser(const std::string& username, const std::string& password) — метод для авторизации пользователя.
void sendMessage(User* sender, User* recipient, const std::string& content) — метод для отправки сообщения конкретному пользователю.
void sendMessageToAll(User* sender, const std::string& content) — метод для отправки сообщения всем пользователям.
void displayMessages() const — метод для вывода всех сообщений.
User* findUserByUsername(const std::string& username) — метод для поиска пользователя по логину.

файл main.cpp
Главный файл программы, который содержит цикл взаимодействия с пользователем через консоль. Он обрабатывает команды пользователя, такие как регистрация, авторизация, отправка сообщений, просмотр сообщений и выход из программы.
