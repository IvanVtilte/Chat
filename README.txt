Чат-приложение на C++
Участники команды
Участник 1: [Иван]
Описание проекта
Цель проекта — создать простое консольное чат-приложение на C++, которое предоставляет пользователям следующие возможности:

Регистрация новых пользователей.
Авторизация зарегистрированных пользователей.
Обмен сообщениями между конкретными пользователями или рассылка сообщений всем сразу.
Структура проекта
Проект реализует объектно-ориентированный подход с разделением функционала на классы:

User
Отвечает за управление данными пользователя: логином, паролем и именем.

Поля:
std::string username — логин пользователя.
std::string password — пароль пользователя.
std::string name — имя пользователя.
Методы:
User(std::string uname, std::string pass, std::string fullname) — создаёт объект пользователя.
void display() const — выводит информацию о пользователе.
Message
Отвечает за передачу текстовых сообщений.

Поля:
User* sender — отправитель.
User* recipient — получатель.
std::string content — текст сообщения.
Методы:
Message(User* sender, User* recipient, const std::string& msg) — создаёт сообщение.
void display() const — выводит сообщение.
Chat
Управляет пользователями, их регистрацией, авторизацией и сообщениями.

Поля:
std::unordered_map<std::string, User*> users — хранение пользователей по логину.
std::vector<Message> messages — хранение всех сообщений.
Методы:
void registerUser(const std::string& username, const std::string& password, const std::string& name) — регистрация пользователя.
User* authenticateUser(const std::string& username, const std::string& password) — авторизация.
void sendMessage(User* sender, User* recipient, const std::string& content) — отправка сообщения конкретному пользователю.
void sendMessageToAll(User* sender, const std::string& content) — рассылка сообщения всем пользователям.
void displayMessages() const — вывод всех сообщений.
User* findUserByUsername(const std::string& username) — поиск пользователя по логину.
main
Главный файл программы. Управляет взаимодействием с пользователем через консоль:

Обрабатывает команды для регистрации и авторизации.
Обеспечивает функционал отправки сообщений.
Предоставляет возможность просматривать историю сообщений.
Завершает программу по команде пользователя.
Идея реализации
Приложение построено на основе объектно-ориентированного подхода, где:

Каждый класс отвечает за определённый аспект работы чата (пользователь, сообщения, управление процессом).
Класс Chat является связующим звеном, организуя взаимодействие между пользователями и их сообщениями.
Консольный интерфейс предоставляет пользователям интуитивно понятный способ управления через команды.
